# HypoScore Mini App — экран «Отчёт» (скаффолд Фазы 5)

Локальный фронтенд-скаффолд флагманского экрана Telegram Mini App: красивый
графитовый отчёт по гипотезе из mock-данных — итоговый балл, радар по 14
критериям, карточки критериев с раскрытием, риски и источники.

Это **только скаффолд Фазы 5**: данные пока образцовые (`src/mockData.ts`),
Telegram SDK не подключён (тема — заглушка, светлая графитовая). Реальные данные
придут из бота через `initData` / API, а тёмная тема — из `themeParams`.

## Как запустить

Нужен Node.js 18+.

```bash
cd hyposcore-miniapp
npm install
npm run dev
```

Открыть адрес из вывода Vite (по умолчанию http://localhost:5173).
Должен появиться мобильный экран отчёта (ширина 360–430) с радаром на 14 осей.

Сборка продакшн-бандла:

```bash
npm run build
npm run preview
```

## Переменные окружения

Задаются в `.env.development` / `.env.production`:

- `VITE_API_BASE` — базовый URL API HypoScore, без завершающего слэша.
- `VITE_BOT_USERNAME` — username бота для deep-link'ов `t.me/<bot>?start=cmd_*`
  (кликабельные команды и оплата в «Кабинете»). Публичная константа, не секрет.
  По умолчанию `hyposcore_bot` — менять не нужно.

## Стек

- **React 18 + TypeScript** — UI и типы формы результата (`EvalResult`).
- **Vite** — dev-сервер и сборка.
- **Recharts** — `RadarChart` по 14 критериям, точки и подписи осей окрашены
  в цвета 5 групп.

## Структура

| Файл | Назначение |
| --- | --- |
| `src/theme.css` | Графитовая палитра, шрифты, все стили экрана (+ тёмная тема). |
| `src/mockData.ts` | TS-типы `EvalResult` и образец данных (14 критериев A1–E2). |
| `src/ReportScreen.tsx` | Экран отчёта: герой, радар, группы, критерии, риски, источники. |
| `src/App.tsx` | Корень: отдаёт `ReportScreen` с mock-данными. |
| `src/main.tsx` | Точка входа React. |

## Визуальный язык

Повторяет `docs/DESIGN_REFERENCE.md` и `docs/report_style_preview.html`:

- **Бумага** `#EDEBE3` + сетка 64px, **чернила** `#16171C`, **акцент-графит** `#2B2D33`.
- **Цвета вердикта** (по баллу): 8–10 зелёный `#2ecc71`, 6–7.9 синий `#3fa7ff`,
  4–5.9 жёлтый `#f5a623`, <4 красный `#e74c3c`.
- **5 групп критериев**: A Рынок `#C2790F`, B Конкуренция `#7A5AF8`,
  C Реализуемость `#0E9E83`, D Риски `#E5484D`, E AI `#2AA0C4`.
- **Шрифты** (Google Fonts): Bricolage Grotesque (дисплей), Spline Sans (текст),
  JetBrains Mono (метки).

Инвертированные критерии (риски, сложность, насыщенность) помечены
«↑ выше = лучше», чтобы шкала читалась однозначно.
